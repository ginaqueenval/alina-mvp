Alina is a dark-themed, international creator platform MVP inspired by subscription-based creator products.

## MVP Scope
This repository is a UI/flow MVP for demonstration and review purposes.

## Features (Implemented)
- Dark premium UI (Tailwind CSS)
- Public pages: Home, Explore, Pricing
- Authentication pages: Sign up, Log in, Forgot password, Reset password
- Role selection (Fan / Creator)
- Feed page (UI placeholder)

## Tech Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
